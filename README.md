# questfy_app_mobile

O Questify combina organização pessoal com gamificação leve e adulta. O usuário realiza missões, mantém sua sequência de dias ativos e acumula progresso, criando um hábito sustentável de produtividade. O foco não está em punir falhas, mas em estimular constância, oferecendo mecanismos inteligentes que respeitam a vida real do usuário.
